# Images

This folder is reserved for public-safe showcase assets.

## Screenshots

Place product screenshots in `images/screenshots/` using the filenames referenced in the root `README.md`.

Recommended screenshot list:

- `dashboard-home.png`
- `daily-signals-feed.png`
- `ai-daily-briefing.png`
- `research-papers.png`
- `paper-analysis-workspace.png`
- `paper-analysis-results.png`
- `concept-explorer.png`
- `concept-detail-panel.png`
- `decision-engine.png`
- `github-pulse.png`
- `daily-tech-radar-email.png`
- `source-status-states.png`

Before adding screenshots, remove or blur:

- API keys, tokens, or environment values.
- Private saved research notes.
- Client names or sensitive data.
- Internal-only URLs.
- Any proprietary prompts or implementation details.

## Diagrams

Place exported architecture diagrams in `images/diagrams/` if you want static images in addition to the Mermaid diagrams in the docs.
